# Portfolio — Lucía Mateo

Landing page de portfolio, diseño de marca & identidad visual.

## Estructura

```
index.html                     → página principal
tripzo.html                    → caso de estudio: Tripzo
panaderia-san-ignacio.html     → caso de estudio: Panadería San Ignacio
koru-travel.html               → caso de estudio: Koru Travel
assets/style.css               → estilos compartidos por todas las páginas
README.md                      → este archivo
```

## Ver en local
Abrir `index.html` en el navegador, o usar la extensión **Live Server** de VS Code (clic derecho → "Open with Live Server").

## Completar las páginas de proyecto
Cada página de caso de estudio (tripzo.html, panaderia-san-ignacio.html, koru-travel.html) tiene
secciones marcadas con recuadros punteados ("Espacio para..." / "[ reemplazar con tu texto real ]").
Reemplazá ese contenido por tus textos e imágenes reales:

1. Buscá el bloque `<div class="placeholder-block">` que querés reemplazar.
2. Para texto: cambiá el texto placeholder directamente.
3. Para imágenes: reemplazá el `<div class="placeholder-block">...</div>` por:
   ```html
   <img src="TU_URL_DE_IMAGEN.jpg" alt="Descripción de la imagen" style="border-radius:16px; width:100%;">
   ```
